# Outage Visualizer

Interactive visualizations of major cloud infrastructure outages.

## Features

- Step-by-step animated simulations of outages
- Visual infrastructure diagrams
- Timeline navigation
- Dark mode support
- Responsive design

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- Framer Motion
- Shadcn/ui

## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build
```
